## 💻 Running the Project Locally

To run the main portfolio site or any of the sub-projects locally:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/mounikarayapalli/mounika-portfolio.git
   cd Path-To-Project
   ```

2. **Start a Local Development Server**:
   Using Python:
   ```bash
   python -m http.server 8000
   ```
   Or using Node.js:
   ```bash
   npx serve .
   ```

3. **Open in Browser**:
   Navigate to `http://localhost:8000` to view the website.
