## 🚀 Key Features

* **🎭 Dynamic Light & Dark Mode**: Seamless toggle interface with custom theme variables.
* **📱 Fully Responsive Design**: Built from scratch with vanilla CSS for mobile, tablet, and desktop layouts.
* **🏷️ Interactive Tabbed Interface**: Modern capsule-styled ("pill") tabs in the **About** section for "My Journey", "What I Build", and "My Vision".
* **🗂️ Detailed Project Modals**: Dynamic modals that display overview, technical specifications, and live demonstrations for each project.
* **📄 Integrated Resume Viewer**: Inline PDF reader/download capabilities.
* **✉️ Direct Contact Form**: Fully configured contact form integrated with FormSubmit.

## 🛠️ Tech Stack & Technologies Used

* **Frontend**: HTML5, CSS3 (Vanilla Custom Properties & Grid/Flexbox), JavaScript (ES6+ for interactive modals, theme switching, and scroll animations).
* **Frameworks & Libs**: Bootstrap (used in *TECH DESIRE* subproject), FontAwesome (icons).
* **Styling Principles**: Glassmorphism, CSS keyframe animations, customized HSL colors, responsive typography.

## 🔧 Run & Host Locally

To run the portfolio website locally, use a simple HTTP server (such as Python's built-in module) to avoid CORS issues with PDFs and local files:

1. Clone this repository:
   ```bash
   git clone https://github.com/mounikarayapalli/mounika-portfolio.git
   cd mounika-portfolio
   ```
2. Start the local server:
   ```bash
   python -m http.server 8000
   ```
3. Open your browser and navigate to:
   ```text
   http://localhost:8000
   ```
